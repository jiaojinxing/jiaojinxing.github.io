 NAME

loc1, loc2 - pointers to characters matched by regular expressions (LEGACY)

 SYNOPSIS



#include <regexp.h>

extern char *loc1;
extern char *loc2;

 DESCRIPTION

Refer to regexp().

 APPLICATION USAGE

These variables are kept for historical reasons, but may be withdrawn in a future issue.

New applications should use fnmatch(), glob(), regcomp() and regexec(), which provide full internationalised regular expression functionality compatible with the ISO POSIX-2 standard, as described in the XBD specification, Regular Expressions .
